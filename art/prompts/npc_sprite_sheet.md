# NPC Sprite Sheet Prompt (4x3)

Create a 2D sprite sheet of [NPC NAME] in a cozy watercolor sticker style.

## Required constraints
- TRUE transparent background (no checkerboard pattern)
- NO white sticker outline or glow around edges
- Thin dark brown/sepia ink outline only
- 4 directions: front, back, left, right
- 3 frames per direction: idle, step1, step2
- Consistent scale and baseline alignment
- Soft watercolor shading, warm pastel palette
- No text

## NPC Description
- [Describe clothing, hair, accessories, mood]

## Output
- Canvas: 768×1024
- Grid: 4 rows × 3 columns, equal cell sizes
- PNG with transparency
