# Props + Icon Prompt

Create a set of cozy watercolor props or inventory icons.

## Requirements
- Transparent background (TRUE alpha)
- Clean silhouette with dark brown/sepia ink outline
- Warm pastel palette
- Minimal micro-detail (must read at 64×64)
- No text labels

## Output
- PNG with transparency
- Provide sizes at 64×64 and 128×128
- Keep consistent lighting and outline thickness across icons
