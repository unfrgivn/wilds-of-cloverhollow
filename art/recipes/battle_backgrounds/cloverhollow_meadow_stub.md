# Cloverhollow meadow stub background

## Goal
Create the deterministic placeholder background for Cloverhollow encounters.

## Inputs
- Script: `tools/art/generate_battle_background_stub.gd`
- Palette intent: Cloverhollow meadow (soft blue sky, green meadow, warm path)

## Output locations
- `art/exports/battle_backgrounds/cloverhollow/meadow_stub/bg.png`
- `art/exports/battle_backgrounds/cloverhollow/default/bg.png`
- `game/assets/battle_backgrounds/cloverhollow/meadow_stub/bg.png`
- `game/assets/battle_backgrounds/cloverhollow/default/bg.png`

## Regeneration
Run from repo root:

```
godot --headless --script tools/art/generate_battle_background_stub.gd
```
